Проект представляет собой популярную игру "Крестики-Нолики", написанную на языке Python. Для создания дружелюбного интерфейса использовались две библиотеки: tkinter и customtkinter.

Для запуска игры необходимо скачать или скопировать код программы и открыть его в редакторе.
Обратите внимание, что так как customtkinter не входит в стандартную библиотеку Python, его может потребоваться установить с помощью команды: pip install customtkinter
